# Summary for: **sparseFinalize**

**sparseFinalize** assemble sparse matrix from sparse struct.

Call syntax
S = **sparseFinalize**(Sf, args)
where Sf = sparse struct and args is a list of arguments (see Matlab's
"sparse" for examples).
